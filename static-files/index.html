Static Files Server

This is the index page served from the static-files/ directory.

Directory Structure:

  • hello.html - Test page
  • css/example.css - Example CSS
  • js/example.js - Example JavaScript
  • README.md - Documentation

How to Use:

Place any files in the static-files/ directory and they will be accessible at http://localhost:8081/

Subdirectories are fully supported!
